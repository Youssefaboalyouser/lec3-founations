يعني إيه شروط؟ 🤔
الشروط هي طريقة لاتخاذ القرارات في البرمجة! زي إشارة المرور تماماً:
لو (if) ⬅️ شرط
إذا كانت الإشارة حمراء، توقف!
if (إشارة == حمراء) {
  توقف();
}
وإلا (else) ⬅️ بديل
إذا كانت الإشارة خضراء، تحرك!
else {
  تحرك();
}
الشروط تساعدنا على اتخاذ القرارات المختلفة حسب الموقف! 🚀
